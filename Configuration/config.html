Debrid / Stremio streams

For each library movie or episode, the server asks your Stremio-style addon for a stream list and shows each line as its own playable source (pick a link like in other streaming UIs). Items need an IMDb id on the movie / series, or a TMDB / TVDB id plus an optional TMDB API key below so the server can resolve IMDb via TMDB (no local IMDb/TVDB database is bundled). Playback uses Real-Debrid or TorBox. You are responsible for complying with laws and service terms. Stream discovery (server menu) lets you browse TMDB without library files; it uses the same settings here.


Each entry appears as a separate version/source in the client (capped for performance).
Free key from themoviedb.org — used only to map TMDB/TVDB metadata on your items to IMDb for the stream addon.
